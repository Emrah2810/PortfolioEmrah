Tambien Stage

Overzicht

Dit was mijn 2e grote opdracht bij Tambien. Ik mocht een Library gaan maken voor de Divoom Pixoo16. Ik moest er dus voor zorgen dat er een manier was om de Pixoo16 te besturen vanaf de Laptop ipv telefoon app.

Werkwijze

Er was niet zo veel documentatie voor de Pixoo16. En de documentatie die er was, was niet heel optimaal. Ik heb dus vanuit andere opensource projecten mijn eigen C# Library gemaakt. Hier mee kan ik de basis dingen doen op de Pixoo16. Zonder dat ik de app ervoor gebruik.

Ik moest eerst de POST request maken. Ik moest eerst de juiste dingen aanroepen voor de request. Gelukkig had ik een paar voorbeelden van de API. Ik heb dus eerst de POST request gemaakt. Dit was een beetje trial and error. Maar uiteindelijk is het gelukt om de juiste request te maken.
